## インストール
```
git clone https://github.com/JordanCampos20/QRCode.git
```
```
pip install -r requirements.txt
```

```
python2とpython3を使用している場合は、requirements.txtをダウンロードするときにpip3を使用してください
```

## ランニング

```
「main.py」ファイルを実行すると、localhost：5000でサイトが起動します。
```
