## 安裝
```
git clone https://github.com/JordanCampos20/QRCode.git
```
```
pip install -r requirements.txt
```

```
如果你有python2和python3，下載requirements.txt時使用pip3

```

## 跑步

```
運行“main.py”文件，它將在您的 localhost:5000 上啟動該站點
```
