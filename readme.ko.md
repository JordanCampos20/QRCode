## 설치
```
자식 클론 https://github.com/JordanCampos20/QRCode.git
```

```
핍 설치 -r 요구 사항.txt
```

```
python2 및 python3이 있는 경우 requirements.txt를 다운로드할 때 pip3을 사용하세요.
```

## 달리기

```
"main.py" 파일을 실행하면 localhost:5000에서 사이트가 시작됩니다.
```
